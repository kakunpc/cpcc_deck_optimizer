# CPCC Deck Optimizer

Chrome 系ブラウザで [からぱり☆カードコレクション!](https://korimizu.com/CPCC/index.php) のデッキ最適化を補助する拡張機能です。

所持カードと解放済みワークをページ上から読み取り、単体ワーク最適化や全ワーク最適化を実行できます。

## インストール方法

1. このリポジトリをダウンロードまたは clone します。
2. Chrome で `chrome://extensions` を開きます。
3. 右上の「デベロッパーモード」を ON にします。
4. 「パッケージ化されていない拡張機能を読み込む」を押します。
5. ダウンロードまたは clone したこのプロジェクトのフォルダを選択します。

## 使い方

1. ゲームページ `https://korimizu.com/CPCC/index.php` または `https://april.colorfulparty.jp/CPCC/` を開きます。
2. デッキ画面のワークタブを開きます。
3. 画面右上に表示される `Optimizerを開く` を押します。
4. パネル内で `再読込` を押して、所持カードとワーク情報を読み込みます。
5. 次のいずれかを実行します。
   - `全ワーク最適化`: 解放済みワーク全体で重複しない組み合わせを探索します。
   - 各ワークボタン: 指定ワークだけ最適化します。
6. 結果表示から必要に応じてデッキのハイライトや自動セットを行います。

## 主な機能

- 所持カードの自動読み取り
- ワークごとの最適デッキ計算
- 全ワークをまたいだ重複なしデッキ計算
- 最適候補カードのハイライト
- 最適化結果の自動セット

## 注意事項

- 現在は生徒会室とたまり場の場効果を無視する前提です。
- オプション `%` は「そのカードの最終 Power にそのまま加算する」形で仮実装です。
- 実ゲームと 1〜2 点ずれる場合は `content.js` 内の `evaluateDeck()` を調整してください。
- ゲーム側の DOM 構造が変わった場合は `parseOwnedCardsRobust()` 付近の調整が必要です。
